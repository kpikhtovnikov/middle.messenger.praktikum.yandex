### Самостоятельный проект "Чат"

# Сборка проекта
npm run build
# Сборка и запуск статического сервера на Express
npm run start

# Запуск проекта в dev режиме
npm run dev

# Eslinter
npm run lint:ts

# Проверка на стилистические и типовые ошибки
npm run lint:style

# Запуск docker
npm run docker

# Запуск docker
npm run test
